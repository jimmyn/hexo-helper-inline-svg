# hexo-helper-inline-svg

Hexo helper that inlines svg image


## Install

``` bash
$ npm install hexo-helper-inline-svg --save
```

## Usage

To count post content words, add in your theme:

```html
<%- inline_svg('/images/example.svg') %>
```
